# **App Name**: Tricks Land Scoreboard

## Core Features:

- Scoreboard Display: Display scoreboards for each class with student names and scores.
- Admin Authentication: Allow admins to view the dashboard only using login credentials (username 'YoussefAhmed', password 'AdminTricksLand123$').
- Admin Dashboard: Admin dashboard to add/remove students, and assign grades.
- Admin Management: Enable admins to create new admin accounts with assigned user names, and passwords.
- Session Scheduling: Implement a modern calendar to allow admins to choose session days.
- Multi-Language Support: Allow multi-language support for Arabic and English.
- AI Encouragement Generator: Provide a motivational message for each student using AI, tailored based on their class and/or progress in Tricks Land. This feature can make use of an AI tool to generate encouraging words that relate to specific subject matter.

## Style Guidelines:

- Primary color: Cheerful orange (#FF8C00) to stimulate optimism and creativity.
- Background color: Light orange (#FFF2E0), almost white, to not distract.
- Accent color: Sky blue (#4680FF) to pop up certain parts of the site, like buttons.
- Headline font: 'Belleza', sans-serif, to lend personality to headings
- Body font: 'Alegreya', serif, for an elegant, intellectual feel in paragraphs
- Use custom-designed icons representing different assessment criteria (e.g., discipline, punctuality, engagement).
- Ensure a clear and intuitive layout with prominent display of scores and admin options.
- Incorporate subtle, delightful animations to enhance user engagement.
